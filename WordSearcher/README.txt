[General Info]

This program was created using SFML - Simple and Fast Multimedia Library

[Runtime Info]

The executable for this project must have the following files within its directory to successfully run:

sfml-graphics-d-2.dll
sfml-system-d-2.dll
sfml-windows-d-2.dll

In addition, several elements such as images, file paths, fonts, etc... are stored in the Resources folder

[Build Info]

To compile this program, visit https://www.sfml-dev.org/download.php to download the appropriate libraries and files for your compiler

At minimum, you will need the appropriate library files, dll files, and SFML include files
